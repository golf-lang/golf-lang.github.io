Gliimly is a programming language for web services. It includes compiler, application server and tools for a complete web development platform. Gliimly is simple, memory-safe, high-performance, built with industry-standard Free Open Source libraries, extensible, and licensed under Apache 2 Free Open Source.

Install Gliimly. See Documentation. Source at github.com (note version comments are the same as their numbers; each change log is described on Gliimly blog).

Current version is 142. Check Blog for change log.
